NAT (Network Address Translation) : 사설 IP가 인터넷 망을 사용하기 위해서 
   패킷의 Source Address를 공인 IP Address로 변환해서 인터넷망으로 내보내야 한다. 

IP Address = Network ID + Host ID
	70.12.113.130	
	255.255.255.0

Host : 네트워크의 장치 (라우터, 스위치, 프린터, 컴퓨터....)
